# 試験メタ情報・公式ソースへのリンク

> **関連 Issue**: [#62](https://github.com/junichi-muraoka/jcsqe-study-app/issues/62)

## 1. 本アプリの立ち位置

- 本リポジトリのアプリは **学習支援** を目的とした非公式ツールです。
- 収録の問題文・解説は **公式の過去問ではなく**、主催者が公開する試験問題の転載ではありません。出題範囲の理解に **SQuBOK Guide 第3版** に沿って独自に作成しています。
- **合格の保証**や **出題の予想** をするものではありません。試験の出題形式・合格基準・日程は **必ず主催者の最新情報**で確認してください。

## 2. 公式情報（主催：日本科学技術連盟 JUSE）

| 内容 | リンク（例） |
|------|----------------|
| JCSQE（試験・申込などの案内） | [juse.jp / JCSQE 試験情報](https://www.juse.jp/jcsqe/exam/) |
| JCSQE 資格制度のトップ | [juse.jp / JCSQE](https://www.juse.jp/jcsqe/) |

※ URL や試験要項は変更されることがあります。**受験前に必ず公式サイトで再確認**してください。

## 3. 出題範囲の参照（SQuBOK）

- 本アプリの「対応範囲」は **SQuBOK Guide 第3版** の構成に合わせています。
- 書籍の購入・最新版の案内は [JUSE の JCSQE 関連ページ](https://www.juse.jp/jcsqe/) や書店の情報を参照してください（版・刷次は試験要項と照合すること）。

## 4. README の表との関係

[README.md](../README.md) の「JCSQE試験について」の表（出題数・時間・合格ライン等）は、**学習時の目安**として記載しています。実際の試験では **公式の試験要項・問題冊子**が優先されます。

## 5. 免責の再掲

README の免責と同趣旨です。試験制度・著作権・商標は主催者および権利者に帰属します。
