# README 用ビジュアル

GitHub の [README.md](../../../README.md) から参照する画像を置くフォルダです。

| ファイル | 用途 |
|----------|------|
| `banner.svg` | リポジトリ冒頭のヒーローバナー（ベクター・テキストは SVG 内） |
| `study-flow.svg` | 学習の流れ（5章 → クイズ → 選択肢別解説）の概念図 |

## 実アプリのスクリーンショットを足すとき

1. 本番またはローカルで画面をキャプチャする（個人情報・学習データに注意）。
2. 例として `screenshot-home.png`、`screenshot-quiz.png` など **意味の分かる名前**でこのフォルダに保存する（PNG または WebP 推奨）。
3. ルートの `README.md` に  
   `![ホーム画面](docs/images/readme/screenshot-home.png)`  
   のように追記する。幅を揃えたい場合は HTML で `<img width="720" ...>` も可。

SVG はリポジトリサイズを抑えつずつ拡大表示に強いです。UI の見た目は **実キャプチャ**の方が信頼感が出ます。バナーは雰囲気用、実画面は別ファイルで足す運用がおすすめです。
